Classgrid Classroom 2.0

What Is Classgrid?

Classgrid is an AI-powered intelligent classroom ecosystem that combines structured academic management, real-time collaboration, and embedded artificial intelligence into one unified learning platform.

It is not just a learning website. It is a smart academic infrastructure designed for modern education.

Our Vision

Our vision is to build a unified digital academic environment where:

  • Faculty manage classrooms with intelligent control
  • Students receive instant AI-assisted support
  • Learning becomes structured, interactive, and adaptive
  • Artificial intelligence works inside the academic workflow

We aim to transform traditional online classrooms into AI-native learning ecosystems.

Classgrid Classroom

Classgrid Classroom provides:

  • Role-based access (Faculty & Students)
  • Secure authentication (Manual + Google OAuth + GitHub + LinkedIn)
  • Real-time classroom interaction
  • Approval-based or code-based class joining
  • Live chat inside each classroom
  • Structured notes, announcements, and quizzes
  • Student note-sharing system
  • AI-powered doubt resolution
  • Intelligent classroom monitoring

Unlike traditional LMS platforms, AI is embedded directly into the system architecture.

AI-Powered Learning Intelligence

Classgrid integrates a multi-model AI system:

  • Primary AI Model: Groq (LLaMA)
  • Fallback Model: Gemini Flash
  • Automatic AI routing for reliability
  • Real-time doubt solving
  • Quiz generation from notes
  • Learning material summarization
  • Academic assistance within the classroom

AI operates as a classroom copilot ‚Äî not as a separate chatbot.

Real-Time Infrastructure

Classgrid is built with a scalable architecture:

  • Node.js + Express backend
  • MongoDB for persistent data storage
  • Supabase for real-time updates
  • Secure authentication systems
  • Fault-tolerant AI fallback routing

This ensures reliability, scalability, and performance.

Secure & Responsible Platform

Classgrid provides secure access through:

  • Email verification-based account creation
  • Self-set password after verification
  • Google, GitHub, and LinkedIn OAuth login
  • Privacy-focused data handling
  • Secure authentication standards

User data is handled responsibly using modern security practices.
